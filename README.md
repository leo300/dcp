# Discount PRO

A simple Progressive Web App (PWA) for quickly calculating discounted prices.

Discount PRO is lightweight, mobile-friendly, and works completely offline once installed. It's perfect for shoppers, retail staff, sales assistants, and anyone who needs to calculate discounts in seconds.

## Features

* 💰 Instant discount calculations
* 📉 Enter any discount percentage (0–100%)
* 🌍 Multiple currency support

  * NOK
  * EUR
  * USD
* 📱 Mobile-friendly interface
* 🌙 Light and Dark theme support (ready for styling)
* ⚡ Works offline as a Progressive Web App (PWA)
* 🚀 Fast and lightweight
* 🔒 No accounts or cloud storage required

## How It Works

1. Enter the original price.
2. Enter the discount percentage.
3. Select your preferred currency.
4. The discounted price is calculated instantly.

## Install as a Mobile App

### Android (Chrome)

1. Open the app in Google Chrome.
2. Tap the **⋮** (three-dot menu).
3. Tap **Install App** or **Add to Home Screen**.
4. Confirm the installation.

The app will appear on your Home Screen and can be used like a normal app.

### iPhone & iPad (Safari)

1. Open the app in Safari.
2. Tap the **Share** button.
3. Select **Add to Home Screen**.
4. Tap **Add**.

The app is now available from your Home Screen.

## Offline Support

Discount PRO uses a Service Worker, allowing it to work even when you don't have an internet connection (after the first visit).

1. Download or clone this repository.

4. Start calculating discounts instantly.
